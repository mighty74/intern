デベロッパーツールの使い方

デバッグ練習

  1. 画面上のどこかで右クリック
  2. 検証を選択
  3. 出てきた画面最上部のSourceをクリック
  4. 左側に debug.js が出てくるのでクリック
  5. console.log("クリック回数：" + count); にブレークポイントを貼る
    (行番号をクリックして数字が青く選択されればOK)
  6. 下のボタンをクリックしてみましょう
  8. Consoleに count と入力し、Enterキーを押しましょう
  9. 整数が表示されることを確認
  10. 処理をひとつ進めましょう
  11. Consoleに クリック回数: X と表示されていることを確認しましょう
  12. 処理を最後まで進めましょう
  13. ポップアップが出てくることを確認

レイアウトの確認

  1. 画面上のどこかで右クリック
  2. 検証を選択
  3. 左上の矢印をクリック
  4. その状態で各htmlタグ要素の上にカーソルを合わせてみよう
  5. その状態で下の箱をクリック
  7. .debug-box の{}の中にcssの記載をしてみよう
  8. border-style: solidのチェックを外す、background-color: #006400 を追記
  9. 枠線が無くなって、背景が緑になればOKです
※カーソルを動かすごとに色々出てくるのが嫌な場合は始めに押した矢印をクリックすれば元に戻ります。
